<h2 align="center"><i>Flutter Based Mobile App</i></h2>

<p>A modern Android application that provides real-time weather information for any location using a weather API.</p>
 <h3>Features</h3>
 <ul>
   <li>Get current weather data (temperature, humidity, wind speed, etc.) for your city.</li>
   <li>Search for weather in any location worldwide.</li>
   <li>Simple and clean user interface.</li>
   <li>Dynamic weather updates using live API integration.</li>
   <li>Error handling for invalid locations or network issues.</li>
 </ul>

 <h3>Techniques</h3>
 <ul>
   <li><b>Language:</b> Flutter / Dart</li>
   <li><b>Platform:</b> Android Studio</li>
   <li><b>API:</b> WeatherApi API</li>
 </ul>
